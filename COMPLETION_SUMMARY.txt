🚀 HOW TO GET STARTED (3 SIMPLE STEPS)

STEP 1: GET YOUR CREDENTIALS (5 minutes)
   • Supabase:  https://app.supabase.com → Settings → API
   • Vercel:    https://vercel.com → Settings & Project Settings
   • Google:    https://aistudio.google.com/app/apikey (optional)

STEP 2: ADD GITHUB SECRETS (3 minutes)

   • Or quick: GitHub → Settings → Secrets and variables → Actions

   $ git commit -m "Add GitHub Actions"
   $ git push origin main
   
   Then check:
   • GitHub: Repository → Actions tab
   • Vercel: Dashboard → Deployments
   • Domain: Visit your custom domain ✨

🔐 DATABASE CREDENTIALS SYSTEM
════════════════════════════════════════════════════════════════════════════════

Your database will work like this:

   GitHub Secrets (encrypted)
        ↓
   GitHub Actions (reads safely)
        ↓
   Create .env.local (build time)
        ↓
   Vite Build (compiles in)
        ↓
   Vercel Deploy (with env vars)
        ↓
   Your App (connects to database)
        ↓
   Supabase (serves data)
        ↓
   ✅ Live with database!
